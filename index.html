Byte Knight Repository

Custom software repository for Debian (APT) and Fedora (RPM) based systems. This is a Linux repository where I publish my open-source projects along with their source code. Everything here is licensed under the GNU General Public License v3.0. Maintained by Neuwj.

Installation

Select your package manager to add the repository and install packages:

Debian / Ubuntu

bash 
# Download the script from the Releases section and run it with this command
chmod +x "RepositoryInstaller.sh" && ./"RepositoryInstaller.sh"
bash 
# Manual install
# Add the GPG key for secure package verification
curl -fsSL https://neuwj-00.github.io/byte-knight/keys/public.key | sudo gpg --dearmor -o /usr/share/keyrings/byte-knight-keyring.gpg

# Add the official repository to your APT sources
echo "deb [signed-by=/usr/share/keyrings/byte-knight-keyring.gpg] https://neuwj-00.github.io/byte-knight/debian stable main" | sudo tee /etc/apt/sources.list.d/byte-knight.list

# Update package lists and install your desired tools
sudo apt update
sudo apt install pacforge repoforge ftr warpcplus

RHEL / Fedora (DNF)

bash 
# Add the Repository configuration
echo "[byte-knight]
name=Byte Knight Repository
baseurl=https://neuwj-00.github.io/byte-knight/fedora/\$basearch/
enabled=1
gpgcheck=1
gpgkey=https://neuwj-00.github.io/byte-knight/keys/public.key" | sudo tee /etc/yum.repos.d/byte-knight.repo

# Update and Install
sudo dnf update && sudo dnf install pacforge

Available Packages

  • pacForge: Automated .deb and .rpm package generator featuring built-in license and certificate support.
  • repoForge: Automated repository indexing and publishing utility.
  • ftr: A lightweight CLI utility for generating structured file and directory tree outputs.
  • warpcpp: An advanced, C++ written terminal interface for Cloudflare's warp-cli.
