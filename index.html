Byte Knight Repository

Custom software repository for Debian (APT) and Fedora (RPM) based systems. This is a Linux repository where I publish my open-source projects along with their source code. Everything here is licensed under the GNU General Public License v3.0. Forked and maintained by Neuwj.

Installation

Select your package manager to add the repository and install packages:

Debian / Ubuntu (APT)

bash 
# Add the GPG Key
wget -qO- https://neuwj-00.github.io/byte-knight/keys/public.key | sudo gpg --dearmor -o /etc/apt/trusted.gpg.d/byte-knight.gpg

# Add the Repository to your list
echo "deb [arch=amd64] https://neuwj-00.github.io/byte-knight/debian stable main" | sudo tee /etc/apt/sources.list.d/byte-knight.list

# Update and Install
sudo apt update && sudo apt install pacforge

RHEL / Fedora (DNF)

bash 
# Add the Repository configuration
echo "[byte-knight]
name=Byte Knight Repository
baseurl=https://neuwj-00.github.io/byte-knight/fedora/\$basearch/
enabled=1
gpgcheck=1
gpgkey=https://neuwj-00.github.io/byte-knight/keys/public.key" | sudo tee /etc/yum.repos.d/byte-knight.repo

# Update and Install
sudo dnf update && sudo dnf install pacforge

Available Packages

  • pacForge: Automated .deb and .rpm package generator featuring built-in license and certificate support.
  • repoForge: Automated repository indexing and publishing utility.
  • ftr: A lightweight CLI utility for generating structured file and directory tree outputs.
  • warpcpp: An advanced, C++ written terminal interface for Cloudflare's warp-cli.
